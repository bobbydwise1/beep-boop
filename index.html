Beep Boop Converter

This program will ask you for a positive integer. It will then create a sequence of numbers starting from 0 up to your number. The output display follows some simple rules:

  • If a number contains the "0" character, it will be replaced with "Beep!".
  • If a number contains the "1" character, it will be replaced with "Boop!".
  • If a number is divisible by 3, it will be replaced with "I'm sorry Dave. I'm afraid I can't do that.".
